## Notas de Desarrollo
- Se utiliza Laravel 10.x
- Base de datos MySQL
- Frontend con Bootstrap 4
- DataTables para tablas dinámicas
- Select2 para selectores avanzados
- SweetAlert2 para notificaciones
- Spatie para roles y permisos
- Laravel Breeze para autenticación
